# DevOps-Studio ローカルLLM運用に関する配置場所

## 管理物
- infra/llm/artifacts/:
  - LLM運用のテンプレ（プロンプト雛形、レビュー観点、チェックリスト）
  - 成果物（テンプレや設計メモ） ※ 公開用に加工済みのもの

## 管理しないもの（.gitignore）
- infra/llm/private/ : 自分用の運用ルール本文・メモ（モデル切替・レビューゲート・禁止事項・事故対応等）
- infra/llm/logs/    : 会話ログ・入出力ログ（時系列）
- infra/llm/runs/    : 作業単位の生のチャットログ、会話のコピペ、スナップショット（後で artifacts に昇格することがある）

方針: 公開して問題ない形に加工したものだけを artifacts として別途Git管理する。
